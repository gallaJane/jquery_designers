Manipulating the DOM

One awesome benefit of jQuery is being able to change the DOM. There are several types of ways we can manipulate the DOM with jQuery:

  • Inside
  • Outside
  • Around
  • Removal
  • Replacement
DOM Methods Reference
Method Description Documentation
.text() If used without an argument, .text() returns an element's text without the HTML tag. It also accepts a string value to set an element's text, or a function. .text()
.html() If used without an argument, .html() returns an element's HTML. It also accepts a string value to to replace an element's HTML, or a function. .html()
.append() Inserts content as the last child of the selected element(s). Accepted arguments: DOM elements, HTML strings, arrays of elements, and functions. .append()
.prepend() Inserts content as the first child of the selected element(s). Accepted arguments: DOM elements, HTML strings, arrays of elements, and functions. .append()
.before() Inserts content before each selected element(s). Accepted arguments: DOM elements, HTML strings, arrays of elements, and functions. .before()
.after() Inserts content after the selected element(s). Accepted arguments: DOM elements, HTML strings, arrays of elements, and functions. .after()
wrap() Wraps an HTML element around each of the selected elements. Accepted arguments: DOM elements, HTML strings, arrays of elements, and functions. wrap();
wrapAll() Wraps an HTML element around the all of the selected elements. Accepted arguments: DOM elements, HTML strings, arrays of elements, and functions. wrapAll();
.remove() Removes the selected element(s) from the DOM, and any elements inside of it. .remove()
.unwrap() Removes the parents of selected element(s) from the DOM, and any elements inside of it. .unwrap()
.replaceAll() Uses the selected element(s) to replace the target element(s). Accepts a selector, jQuery Object, DOM elment, or array. .replaceAll()
.replaceWith() Replace the selected element(s) with provided content and return the set of removed elements. .replaceAll()
